## Running Locally
1. Clone the repository:
    ```bash
    git clone https://github.com/mouleshleo/mouleshleo.github.io.git
    ```

2. Navigate to the project directory:
    ```bash
    cd mouleshleo.github.io
    ```

3. Open the `index.html` file in a web browser:
    ```bash
    open index.html
    ```
