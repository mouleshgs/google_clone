## Running Locally
1. Clone the repository:
    ```bash
    git clone https://github.com/mouleshgs/google_clone.git
    ```

2. Navigate to the project directory:
    ```bash
    cd google_clone
    ```

3. Open the `index.html` file in a web browser:
    ```bash
    open index.html
    ```
